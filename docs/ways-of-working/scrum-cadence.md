# Scrum Cadence (Minimal)

## Sprint length
- 2 weeks (default)

## Events
### Sprint Planning (60–90 min)
- Define Sprint Goal
- Select stories based on capacity
- Confirm acceptance criteria and dependencies

### Daily (10–15 min)
- Yesterday / Today / Blockers
- Raise blockers immediately

### Sprint Review (30–60 min)
- Demo working increments
- Review KPIs / outcomes
- Align next priorities

### Retrospective (30–45 min)
- What went well
- What to improve
- Concrete actions (max 3)
